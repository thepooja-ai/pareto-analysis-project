**Sales Data Analysis & Pareto Dashboard**
This project presents a complete  end-to-end data analysis workflow using Google Sheets. It includes data cleaning, transformation, pivot-based analysis, Pareto analysis, and an interactive dashboard.
The objective of this project of this project is to derive meaningful business insights from raw sales data and identity key revenue contributors.
**Dataset Description**
The dataset consist of:
Customer information
Product details
Sales transactions
Region and category data
**Tools Used**
Google Sheets
Data Cleaning & Transformation
VLOOKUP/Lookup Functions
Pivot Tables
Sorting & Ranking
Cumulative Calculations
Pareto Analysis(80/20 Principle)
Dashboard Design
**Analysis Performed**
Data Preparation
Cleaned raw dataset
Mapped customer and product details using lookup functions
Structured final dataset for analysis
Pivot Table Analysis
Sales by Customer
Sales by Region
Sales by Category
Time-based sales trends
Pareto Analysis 
Calculated cumulative sales
Derived cumulative percentage
Identified top contributors based on revenue share
**Key Insights**
A single customer contributes approximately 90% of total revenue, indicating extreme revenue concentration.
The standard Pareto principle (80/20 rule) does not apply in this dataset due to skewed distribution. 
Revenue distribution is highly uneven, with minimal contribution from remaining customers.
The business shows high dependency on one major customer, posing potential risk.
There is a strong opportunity to improve revenue diversification by targeting smaller customers.
**Dashboard Features**
KPIs cards (Total Sales, Key Metrics)
Sales distribution visuals
Customer performance analysis
Clean and interactive layout
**Conclusion**
This project highlights how data analysis can uncover critical business insights. The findings reveal a highly concentrated revenue structure, emphasizing the importance of customer diversification for long-term stability.
